# Customer Segmentation Project
Data science project analyzing customer purchasing behavior using K-Means clustering and RFM metrics.

# Project Overview
Clustered 50K+ customers into 4 actionable segments:
Premium (High-value loyal customers)
At-Risk (Declining spenders)
New (Recent purchasers)
Inactive (Dormant accounts)

# Key Methods
RFM Analysis: Recency, Frequency, Monetary scoring
K-Means Clustering with elbow method optimization
PCA for dimensionality reduction
Customer profiling with visualizations

# Results Delivered
text
Premium Segment: 12% of customers, 45% of revenue
At-Risk: 18% customers, targeted win-back campaigns
